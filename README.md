# cultural-heritage-server
基于SpringBoot+Vue前后端分离的贵州非遗网站，系统包含管理员、用户角色，功能如下： 管理员：首页统计（非遗统计、传承人统计、体验预约统计）；用户管理；非遗种类；非遗管理；传承管理；文化体验（体验路线管理、体验预约）；非遗商城（商品分类、商品管理、订单管理、商品评论）、活动管理；活动报名；系统留言；系统公告个人信息；密码修改。 用户：注册、登录系统；首页；非遗项目（分类、列表、详情、收藏）；传承人（分类、详情）；文化体验（详情、预约购票）；非遗商城（分类、商品详情、加入购物车、下单、评论）；非遗活动（详情、报名）；系统公告。用户还具有个人中心功能，功能包括：非遗收藏、文化体验、我的订单、商品评论、活动报名、系统留言、个人信息、修改密码。
